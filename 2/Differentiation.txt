Производная функции — это мера того, как изменяется значение функции при изменении её аргумента. Простыми словами, 
производная показывает скорость изменения функции в каждой точке.

Производная функции — наклон касательной к исходной функции в определенной точке. 

Производная функции — это предел отношения приращения функции к приращению аргумента (секущая линия), когда приращение стремится к нулю. 


Как найти производную: основные правила с примерами (нотация Лагранжа)

Правило 1. Производная степени

    Если y = x^n, где n — число, то
    y' = nx^{n-1}.

    Пример:
    y = x^3, тогда
    y' = 3x^2.

Правило 2. Производная суммы равна сумме производных:

    (y_1 + y_2)' = y_1' + y_2'.

    Пример:
    y = x^2 + 3x,
    y' = 2x + 3.

Правило 3. Производная функции, умноженной (или деленой) на функцию:

    (f * g)' = f' * g + f * g'.

    Пример:
    y = x^2 * sin x,
    y' = 2x * sin x + x^2 * cos x.
    
    (f / g)' = (f' * g - f * g') / g^2

Правило 4. Производная сложной функции (цепное правило)

    Если y = f(g(x)), то
    y' = f'(g(x)) * g'(x).

    Пример:
    y = cos(2x),
    y' = -sin(2x) * 2 = -2 sin(2x).

Правило 5. Производная любой константы равна нулю.


Таким образом, чтобы найти производную функции, применяют правила дифференцирования с использованием известных формул и простых операций 
над функциями. Это позволяет быстро находить производные даже сложных выражений без вычисления пределов вручную.

Частная производная — это производная функции нескольких переменных по одной из этих переменных, при условии, 
что все остальные переменные считаются постоянными.
Частная производная ∂f/∂x в точке — это наклон касательной к графику функции в направлении оси x (при фиксированном y).

Правила вычисления частных производных похожи на обычные правила дифференцирования функции одной переменной. 
Главное — учитывать, что при дифференцировании по одной переменной остальные считаются константами.
(нотацию Лейбница удобнее использовать для записи частных производных):

Пример:
    f(x,y) = x^2y + 3xy^2,
    
Частная производная по x (при фиксированном y):
    ∂f/∂x = 2xy + 3y^2.
    
Частная производная по y (при фиксированном x):
    ∂f/∂y = x^2+6xy.
    
    
    Частные производные высших порядков:
Можно находить вторые, третьи и т.д. частные производные:

    ∂^2f/∂x^2 — вторая производная по x
    ∂^2f/∂x∂y — сначала по y , потом по x
    ∂^2f/∂y∂x — сначала по x , потом по y
    
    

Градиент — это вектор, состоящий из частных производных функции потерь (ошибки) по параметрам модели. 
Градиент есть не что иное, как совокупность частных производных по каждой из независимых переменных. Его можно назвать «полной производной».
Он обозначается через греческую букву «набла», ∇ или оператор Гамильтона.
Он указывает направление наибольшего возрастания функции потерь, а задача алгоритма оптимизации — двигаться в направлении, 
противоположном градиенту, чтобы минимизировать ошибку модели.

Определение градиента:
Градиент функции потерь L по параметрам модели w вычисляется как вектор частных производных 

    ∇wL=[∂L/∂w1, ∂L/∂w2, … , ∂L/∂wn]
    
Метод градиентного спуска — это способ нахождения локального минимума функции в процессе движения в направлении антиградиента.
При этом, если мы сдвинемся на всю величину градиента, то «перескочим» через минимум. Именно поэтому нам нужен коэффициент скорости обучения.



Промежуточные выводы:

    1. градиент показывает скорость изменения многомерной функции по каждому из измерений (по каждой из переменных);
    2. если «заморозить» все независимые переменные кроме одной и сделать срез, то наклон касательной к кривой среза будет значением частной производной по этому измерению;
    3. градиент показывает направление скорейшего подъёма (возрастания) функции; двигаясь в обратном направлении, мы придем к минимуму функции.
